Unity（简称：Unity3D / Unity2D，简称 U3D）是目前 由丹麦Unity Technologies 开发的最受欢迎的多平台免费游戏引擎（主要用 C，C ++ 和 C＃编写，于 2005 年首次发布）。具有 C＃和 JavaScript 编程经验的用户可以更快地入门。

作为跨平台且高度集成的游戏开发工具（即，专业游戏引擎的全面集成），类似于 Adobe Director，Blender，Virtools 或Torque Game Builder等等在交互式游戏开发环境中，Unity 从一开始就被设计为易于使用的产品。借助它，用户可以轻松创建多种类型的交互式内容，例如 3D 视频游戏，建筑可视化，实时 3D 动画等。除了简单易用的用户界面设计优势外，Unity 的功能还包括集成的可视化编辑器，跨平台发行版（允许您将游戏作品快速发布到 Windows，Mac OS X，Wii，XBOX，PlayStation，Android， iPhone，iOS，Windows Phone，WebGL和许多其他平台），地形编辑器，着色器，脚本编辑器，版本控制等。

最后，如果让我选择了手机游戏，网页游戏，单机游戏和 PC 游戏的发展更有前途的 3D 和 2D 游戏引擎，我想谈谈团结，而不是类似于著名游戏引擎 GameMaker 工作室 哪个更擅长 2D 游戏开发。需要提醒大家的是，Unity 当然不限于 2D / 3D 游戏的开发。实际上，它也可以用于制作精美的 3D 动画等许多其他类型的交互式内容！

** 适用于 Windows 上的 Unity 2019.x / 2020.x：**

** 带有解锁文件和通用许可文件：**

  1. 首先，删除 C 卷下所有与 Unity 相关的文件夹和文件，例如：
    • C：\ Users \ [用户名 ] \ AppData \ LocalLow \ Unity \
    • C：\ Users \ [用户名 ] \ AppData \ Local \ Unity \
    • C：\ Users \ [用户名 ] \ AppData \ Roaming \ Unity \
  2. 删除所有与 Unity 相关的注册表项，例如：
    • HKEY_CURRENT_USER \ Software \ Unity \ UnityEditor \
    • HKEY_CURRENT_USER \ Software \ Unity \
  3.  从官方网站下载并安装Unity Hub，请勿运行它
  4. 将未锁定的文件 “ app.asar ” 复制到 Unity Hub 的安装文件夹（例如“ *D：\ Program Files \ Unity Hub \ resources \* ”）并覆盖

  6. 将未锁定的文件 “ Unity.exe ” 复制到安装文件夹（例如“ *D：\ Program Files \ Unity \ Editor \* ”）并覆盖

  8. 将许可证文件 “ Unity_lic.ulf ” 复制到 “ *C：\ ProgramData \ Unity \* ” 并覆盖（您可能需要手动创建 “ Unity ” 文件夹）
  9. 运行 Unity Hub，并创建一个新项目以通过许可证验证（而不是立即添加或打开旧项目）
  10. 完毕
