二、将自己的插件发布到服务器上面

  • 脚本建议按照 Editor 和 Runtime 进行分类
  • package 里所有的脚本都需要有一个程序集定义文件 Assembly Definition
  • 程序集之间的引用关系必须正确
  • 根目录必须要有一个 package.json 文件

  1. 因为 Editor 的脚本应该是只在编辑器下生效的，所以我们需要将平台改为只勾选 Editor，否则有可能在打包时出错
