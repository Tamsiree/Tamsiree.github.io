添加菜单

添加系统菜单 (Window) 的菜单项

添加菜单项的快捷键

% 代表 Ctrl 或 CMD
# 代表 Shift
& 代表 Alt
LEFT/RIGHT/UP/DOWN 代表方向键
F1 … F2 代表功能键
HOME, END, PGUP, PGDN 代表对应按键
_[a-z] 代表 A-Z 字符

以上每项快捷键可以组合使用，字符单独使用需要‘_’前缀，组合使用不需要‘_’前缀。快捷键冲突不会有提示，只会有一个被启用。

添加右键上下文菜单，使用内置指定路径

Assets/ 工程视图右键菜单
Assets/Create/ 工程视图右键 Create 菜单
CONTEXT/ComponentName/ 属性面板对应组件的右键菜单
