方法二：申请 Google API

  1. https://cloud.google.com/console
  2. 创建或选择已有项目 $\Rightarrow$ 左侧边栏 API 和 服务 $\Rightarrow$ 凭证
  3. 创建凭证 (类型为 “API 密钥”, 名称随意, 不使用密钥限制, 记住生成的 key)
  4. 再创建一个凭证 (类型为 “OAuth 客户端 ID”, 名称随意, 应用类型选择 “其他”, 记住生成的 “客户端 ID” 和 “客户端密钥”)
  5. 格式填写自己的 API Key
