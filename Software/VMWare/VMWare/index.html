еүҚиЁҖ

VMWare (Virtual Machine ware)жҳҜдёҖдёӘвҖңиҷҡжӢҹPCвҖқиҪҜд»¶е…¬еҸё.е®ғзҡ„дә§е“ҒеҸҜд»ҘдҪҝдҪ еңЁдёҖеҸ°жңәеҷЁдёҠеҗҢж—¶иҝҗиЎҢдәҢдёӘжҲ–жӣҙеӨҡWindowsгҖҒDOSгҖҒLINUXзі»з»ҹгҖӮ
дёҺвҖңеӨҡеҗҜеҠЁвҖқзі»з»ҹзӣёжҜ”пјҢVMWareйҮҮз”ЁдәҶе®Ңе…ЁдёҚеҗҢзҡ„жҰӮеҝөгҖӮеӨҡеҗҜеҠЁзі»з»ҹеңЁдёҖдёӘж—¶еҲ»еҸӘиғҪиҝҗиЎҢдёҖдёӘзі»з»ҹпјҢеңЁзі»з»ҹеҲҮжҚўж—¶йңҖиҰҒйҮҚж–°еҗҜеҠЁжңәеҷЁгҖӮVMWareжҳҜзңҹжӯЈвҖңеҗҢж—¶вҖқиҝҗиЎҢпјҢеӨҡдёӘж“ҚдҪңзі»з»ҹеңЁдё»зі»з»ҹзҡ„е№іеҸ°дёҠпјҢе°ұиұЎж ҮеҮҶWindowsеә”з”ЁзЁӢеәҸйӮЈж ·еҲҮжҚўгҖӮиҖҢдё”жҜҸдёӘж“ҚдҪңзі»з»ҹдҪ йғҪеҸҜд»ҘиҝӣиЎҢиҷҡжӢҹзҡ„еҲҶеҢәгҖҒй…ҚзҪ®иҖҢдёҚеҪұе“Қзңҹе®һзЎ¬зӣҳзҡ„ж•°жҚ®пјҢдҪ з”ҡиҮіеҸҜд»ҘйҖҡиҝҮзҪ‘еҚЎе°ҶеҮ еҸ°иҷҡжӢҹжңәз”ЁзҪ‘еҚЎиҝһжҺҘдёәдёҖдёӘеұҖеҹҹзҪ‘пјҢжһҒе…¶ж–№дҫҝгҖӮе®үиЈ…еңЁVMwareж“ҚдҪңзі»з»ҹжҖ§иғҪдёҠжҜ”зӣҙжҺҘе®үиЈ…еңЁзЎ¬зӣҳдёҠзҡ„зі»з»ҹдҪҺдёҚе°‘пјҢеӣ жӯӨпјҢжҜ”иҫғйҖӮеҗҲеӯҰд№ е’ҢжөӢиҜ•гҖӮ

Oracle VM VirtualBoxе®үиЈ…еўһејәеҠҹиғҪе’Ңе…ұдә«ж–Үд»¶еӨ№зҡ„ж–№жі•

е®үиЈ…еўһејәеҠҹиғҪ

йҖүжӢ©е®үиЈ…еўһејәеҠҹиғҪзҡ„е®үиЈ…еҢ…жңү2з§Қж–№жі•пјҢеҲҶеҲ«еҰӮдёӢпјҡ
з¬¬дёҖз§ҚпјҡжүҫеҲ°и®ҫеӨҮ->е®үиЈ…еўһејәеҠҹиғҪ

з¬¬дәҢз§Қ

  1. жүҫеҲ°и®ҫеӨҮ->зӮ№еҮ»йҖүжӢ©иҷҡжӢҹзӣҳгҖӮ
  2. жүҫеҲ°VirtualBox->VBoxGuestAdditions.isoж–Үд»¶гҖӮ

е…ұдә«ж–Үд»¶еӨ№

дёӢиҪҪең°еқҖпјҡhttp://download.virtualbox.org/virtualbox/

  1. жүҫеҲ°иҸңеҚ•ж ҸдёҠзҡ„и®ҫеӨҮ->йҖүжӢ©е…ұдә«ж–Үд»¶еӨ№->еҲӣе»әе…ұдә«ж–Үд»¶еӨ№(е…ұдә«ж–Үд»¶еӨ№е°ұжҳҜжң¬з”өи„‘е…ұдә«ж–Үд»¶еӨ№пјҢиҜҘж–Үд»¶еӨ№и®ҫзҪ®иҮӘеҠЁжҢӮиҪҪеҲҶй…ҚпјҢиҝҷж ·жҜҸж¬ЎеҗҜеҠЁиҷҡжӢҹжңәе°ұеҸҜд»ҘиҝӣиЎҢе…ұдә«ж–Үд»¶гҖӮ)
  2. йҮҚж–°иҝӣе…ҘиҷҡжӢҹUbuntuпјҢеңЁе‘Ҫд»ӨиЎҢз»Ҳз«ҜдёӢиҫ“е…Ҙпјҡ
    sudo mkdir /mnt/sharing ж–°еҲӣе»әдёҖдёӘжҢӮиҪҪзӮ№пјӣ
    sudo mount -t vboxsf Share /mnt/sharing иҝӣиЎҢжҢӮиҪҪ
    жү§иЎҢдёӢls зңӢдёӢжңүжІЎжңүжҢӮиҪҪжҲҗеҠҹгҖӮ
  3. еҰӮжһңдёҚжғіжҜҸж¬ЎеҗҜеҠЁеҗҺйғҪиҝӣиЎҢжҢӮиҪҪзҡ„иҜқпјҢеҸҜд»Ҙи®ҫзҪ®дёәиҮӘеҠЁжҢӮиҪҪпјҢеҰӮжһңжңүйҖүдёӯвҖңAuto-mountвҖқзҡ„йҖүйЎ№зҡ„иҜқ,еҸҜеңЁ/media/зӣ®еҪ•дёӢзңӢеҲ°дҪ е…ұдә«зҡ„еҗҚеӯ—sf_name;дҫӢеҰӮе…ұдә«зҡ„ж–Үд»¶еӨ№еҗҚеӯ—дёәshareпјҢйӮЈе°ұзңӢеҲ°sf_shareгҖӮ

еңЁжң¬зі»з»ҹе…іиҒ”иҷҡжӢҹжңәдёҠзҡ„зі»з»ҹпјҢиҝӣиЎҢж–Үд»¶е…ұдә«

зӮ№еҮ»жЎҢйқўдёҠзҡ„зҪ‘з»ңеҸіеҮ»йј ж ҮпјҢйҖүжӢ©жҳ е°„зҪ‘з»ңй©ұеҠЁеҷЁпјҢе…¶дёӯзҡ„ж–Үд»¶еӨ№зҡ„и·Ҝеҫ„пјҢдёәе…ұдә«ж–Үд»¶еӨ№зҡ„и·Ҝеҫ„пјҢжңҖеҗҺеңЁз”өи„‘дёӯзҡ„зҪ‘з»ңдҪҚзҪ®пјҢеҮәзҺ°е…ұдә«ж–Үд»¶зҡ„еӣҫж ҮпјҢеҸҢеҮ»иҜҘж–Үд»¶еӨ№пјҢжҳҫзӨәдәҶе…ұдә«зҡ„еҶ…е®№гҖӮ

жңҖеҗҺпјҢе®һзҺ°зҡ„з»“жһңеҰӮдёӢеӣҫпјҡ

VMware жүҖжңүзүҲжң¬ж°ёд№…и®ёеҸҜиҜҒ
